# Statistics Fundamentals

Statistics is the science of **collecting, analyzing, interpreting, and presenting data**.

The main purpose of statistics is to **understand patterns in data and make conclusions about a population using a sample**.

---

# Purpose of Statistics

Statistics helps us:

- Understand **patterns and structure** in data
- **Summarize datasets** using meaningful numbers
- **Visualize data** clearly
- **Make predictions or inferences** about a larger population

---

# Example

Suppose we roll a **six-sided dice 1000 times** and record the outcomes.

Questions we might ask:

- Do all numbers appear roughly the same number of times?
- Does one number appear more often than expected?

If one number appears **significantly more frequently**, the dice might be **biased**.

Statistics helps us determine whether the observed outcome is **random variation or evidence of bias**.

---

# Branches of Statistics

Statistics is divided into two main branches.

```mermaid
flowchart TD
    A[Statistics] --> B[Descriptive Statistics]
    A --> C[Inferential Statistics]

    1. Descriptive Statistics

Descriptive statistics focuses on summarizing and describing the main features of a dataset.

It answers questions such as:

What is the average value?

How spread out is the data?

What values occur most frequently?

Descriptive statistics does not make predictions about a larger population.

Measures of Central Tendency

These describe the center of the dataset.

flowchart TD
    A[Central Tendency] --> B[Mean]
    A --> C[Median]
    A --> D[Mode]

Mean
Average value of all observations.

Median
Middle value when the data is sorted.

Mode
Most frequently occurring value.

Measures of Spread (Dispersion)

These describe how much the data varies.
flowchart TD
    A[Measures of Spread] --> B[Variance]
    A --> C[Standard Deviation]
    A --> D[Range]

Variance – measures how far values deviate from the mean

Standard Deviation – square root of variance

Range – difference between maximum and minimum values

2. Inferential Statistics

Inferential statistics uses sample data to make conclusions about a population.

Since collecting data from an entire population is often impractical, we take a sample and use statistical methods to estimate population characteristics.

flowchart LR
    A[Population] --> B[Sample]
    B --> C[Statistical Analysis]
    C --> D[Inference about Population]



Common Techniques

Hypothesis Testing

Confidence Intervals

Regression Analysis

Sampling Methods

Key Idea

Descriptive Statistics

→ Describes and summarizes the data you already have.

Inferential Statistics

→ Uses sample data to make predictions or conclusions about a larger population.
